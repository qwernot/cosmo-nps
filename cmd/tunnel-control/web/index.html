一键部署节点

          
        

        

          
您可以在任何 Linux 目标服务器上运行以下一键部署指令。脚本会**优先通过 Docker** 启动容器节点；若无 Docker，则会自动**下载二进制并注册为 Systemd 系统服务**自启。

          

            
curl -fsSL "..." | bash

            
          

          

            
💡 部署提示

            
    
              
  • 服务器需要能够通过网络正常访问主控端；
    • 
              
  • 非 Docker 部署方式需要拥有 `root / sudo` 权限以注册系统服务；
    • 
              
  • 如需手动下载对应的 `tunnel-agent` 二进制，请点击此处直接下载。
